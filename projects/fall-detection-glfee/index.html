Fall Detection Algorithm Based on Global and Local Feature Extraction

Bin Li*, Jiangqiao Li, Wang Peng
Pattern Recognition Letters, 2024
IF: 5.1, JCR Q2, CCF C
Paper Code (Coming Soon) BibTeX

Abstract

我们提出了一种基于全局和局部特征提取的跌倒检测算法。该方法结合了深度学习和传统计算机视觉技术， 在保持高准确率的同时实现了实时检测。通过提取视频序列中的全局运动特征和局部姿态特征， 我们的方法能够有效区分跌倒和其他日常活动，适用于智能家居和医疗监护等场景。

Key Features

  • ✅ 高准确率：在多个公开数据集上达到95%以上的准确率

  • ✅ 鲁棒性强：对光照变化、遮挡等情况具有良好的适应性
  • ✅ 易于部署：轻量级模型设计，适合边缘设备部署
